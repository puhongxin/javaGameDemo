以上为该游戏的截图
目前已经实现了如下功能：<br>
1.两个场景的自由切换 <br>
2.人物的鼠标和方向键的移动 <br>
3.按空格键可以与npc进行对话，需要在npc附近触发对话，因为程序中会对主角位置和npc位置进行判断 <br>
4.人物移动的时候加入了碰撞检测，如果遇到障碍物会无法前进，主要的实现手段是将场景中障碍物进行相应的染色处理，然后进行判断 <br>
